\# Modern Workplace Hybrid Lab – Portfolio Summary



\## Objectif

Présenter une synthèse professionnelle du lab Modern Workplace hybride, destinée à :

\- LinkedIn

\- GitHub

\- CV

\- Entretiens techniques

\- Présentations architecte



Ce résumé met en avant les compétences clés, l’architecture mise en place et les technologies maîtrisées.



\---



\# 1. Résumé professionnel



Ce projet démontre la maîtrise complète d’un environnement \*\*Modern Workplace hybride\*\*, incluant :



\- Identité hybride (AD On-Prem + Entra ID)

\- Hybrid Azure AD Join

\- Intune MDM \& Configuration Profiles

\- Autopilot (User-driven, Self-deploying, Pre-provisioning)

\- Packaging Win32 (.intunewin)

\- Security Baselines \& Compliance

\- Zero Trust (Conditional Access, Identity Protection)

\- Microsoft Defender for Endpoint (EDR, TVM, ASR)

\- Automatisation \& standardisation du poste de travail



L’objectif : construire une architecture \*\*cloud-first\*\*, sécurisée, automatisée et alignée avec les bonnes pratiques Microsoft.



\---



\# 2. Compétences démontrées



\## Identité \& Hybridation

\- AD DS + Entra ID

\- Entra Connect (PHS, Device Writeback)

\- Hybrid Azure AD Join

\- MFA, Passwordless, Identity Protection



\## Gestion du poste de travail

\- Intune MDM

\- Compliance Policies

\- Security Baselines

\- Configuration Profiles (Firewall, BitLocker, Windows Update)



\## Déploiement \& Automatisation

\- Windows Autopilot (end-to-end)

\- ESP (Enrollment Status Page)

\- Win32 Packaging (.intunewin)

\- Scripts PowerShell IME



\## Sécurité \& Zero Trust

\- Conditional Access (Require MFA, Require compliant device, Block legacy auth)

\- Microsoft Defender for Endpoint (EDR, TVM, ASR)

\- Device Risk Score + CA

\- Hardening Windows 10/11



\---



\# 3. Architecture mise en place



Le lab repose sur une architecture hybride moderne :



\- \*\*AD On-Prem\*\* pour l’identité locale  

\- \*\*Entra ID\*\* comme autorité d’identité cloud  

\- \*\*Hybrid Join\*\* pour les appareils  

\- \*\*Intune\*\* pour la gestion du poste de travail  

\- \*\*Autopilot\*\* pour le déploiement automatisé  

\- \*\*MDE\*\* pour la sécurité avancée  

\- \*\*Zero Trust\*\* comme modèle de sécurité global  



Cette architecture reflète les standards Microsoft pour les environnements d’entreprise.
